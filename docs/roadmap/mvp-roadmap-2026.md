# QUANTUM LP - MVP開発ロードマップ（2026年Q2リリース）

## 📱 MVP製品ビジョン

**ミッション:** 「5分で作れて、確実に売れるLP」
**ターゲット:** 個人事業主・中小企業（月商100万〜1億円）
